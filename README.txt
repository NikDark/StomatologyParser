!!! Эта инструкция по работе с парсером данных с поисковиков 
bing.com, google.com, yandex.com. mail.ru
по запросу Стоматология !!!

Для работы с парсером у вас должен быть установлен Python >= 3.6

Шаг 1:

    Переместите файл geckodriver.exe в папку
    C:\Users\<Имя пользователя>\AppData\Local\Programs\Python\Python36\Scripts
    (Этот файл нужен для корректной работы парсинга данных с сайта mail.ru)

Шаг 2:

    откройте в консоли папку с парсером и введите
    
    >> pip install .

    Все зависимости установятся

Шаг 3:

    Запустите парсер через главный файл make_all.py командой

    >> python make_all.py
    Дождитесь окончания выполнения программы


Также в комплекте идет уже настроенное виртуальное пространство.
Для его активации, перейдите в папку venv\Scripts и запустите activate.bat в консоли через команду

>> .\activate.bat

и после этого выполните Шаг 3 для запуска парсера
